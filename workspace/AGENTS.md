# Available Agents

This file defines specialized agents and their capabilities.

## Default Agent

The default coding assistant handles general tasks:
- Code review and suggestions
- Bug fixing and debugging
- Feature implementation
- Code refactoring
- Documentation writing

## Code Review Agent

Specialized in reviewing code for:
- Best practices compliance
- Security vulnerabilities
- Performance optimizations
- Code style consistency
- Test coverage

## Debug Agent

Specialized in debugging:
- Analyzing error messages and stack traces
- Tracing issues through code paths
- Suggesting targeted fixes
- Writing test cases to prevent regression

## Refactor Agent

Specialized in code improvement:
- Identifying code smells
- Suggesting design patterns
- Breaking down large functions
- Improving code organization

## Documentation Agent

Specialized in documentation:
- Writing API documentation
- Creating README files
- Documenting code architecture
- Writing inline comments
